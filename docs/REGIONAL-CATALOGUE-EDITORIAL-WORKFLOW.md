# Regional Catalogue Editorial Workflow

1. Locate an official or authoritative temple identity source.
2. Register official name, Tamil name, district and source identifier.
3. Keep changing visitor details separate from stable identity.
4. Review history, legends and architecture field by field.
5. Label devotional tradition separately from historical claims.
6. Add only licensed and attributed photography.
7. Validate the JSON record against the schema.
8. Add the compact record to the regional index.
9. Run route and regression tests.
