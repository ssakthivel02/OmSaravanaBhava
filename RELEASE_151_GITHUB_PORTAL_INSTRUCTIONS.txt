2. Open the extracted folder and select everything inside it.
3. Open the OmSaravanaBhava repository on branch main.

5. Drag the selected extracted contents into the upload area.
6. Confirm there is no Release151, package, output or cleanup parent folder.

8. Leave the lower description blank or optional.
9. Select Commit directly to the main branch.
10. Click Commit changes.
