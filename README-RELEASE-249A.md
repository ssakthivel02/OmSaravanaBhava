# Release 249A — Temple Encyclopedia Foundation

## Dependency
Release 248 must already be live and validated.

## Purpose
Release 249A replaces duplicated temple-page logic with a reusable data-driven encyclopedia engine.

## Core additions
- Temple encyclopedia browser
- Search and filtering
- Structured temple index
- Individual JSON files for all six Arupadai Veedu temples
- Reusable temple detail runtime
- Gallery, facts, festivals, related-content and pilgrimage modules
- Local pilgrimage saving
- Publication-status and image-integrity controls
- JSON Schema for future temple records
- SEO metadata support
- Validation and regression documentation
