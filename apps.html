Installation guidance

When the install button is not available

Chrome / Edge desktop

Open the browser menu or installation icon and choose Install Om Saravana Bhava when offered.

Android browsers

Use the browser menu and choose Install app or Add to Home Screen after loading the site online.

Safari on iPhone or iPad

Use Share and choose Add to Home Screen. Available capabilities depend on the current browser and operating system.

Native app publication checklist

Required before a store badge or package link is displayed

Official listing URL, application ID, publisher/owner, version and release date, supported operating systems, privacy policy, data-safety disclosure, support contact, screenshots with provenance, checksum/signature where applicable, accessibility statement, content/source boundary and confirmation that the listed build matches the reviewed source.
