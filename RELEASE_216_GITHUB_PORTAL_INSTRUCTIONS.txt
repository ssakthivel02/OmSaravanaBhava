UPLOAD
1. Download and extract OmSaravanaBhava-Release-216.zip.
2. In Windows File Explorer enable View > Show > Hidden items.
3. Open the extracted folder.
4. In GitHub open the repository main branch.
5. Choose Add file > Upload files.
6. Drag the CONTENTS of the extracted folder, not the outer folder itself.
7. Preserve all directory paths.
8. Before committing, confirm GitHub displays:
   .github/workflows/static-site-integrity.yml
   .github/workflows/production-smoke.yml
   assets/js/reading-list.mjs
   assets/css/reading-list.css
   reading-list.html
   tools/release_216_validate.py
9. Enter the required commit title in the small upper field.
10. Leave the large description field empty.
11. Commit directly to main.
