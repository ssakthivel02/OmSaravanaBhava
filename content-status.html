Interpretation boundary

  • The Release 231 overlay converts historical generic route labels into the verified canonical status before public discovery and directory rendering.
  • Published must not be read as “complete work” when the page itself states that only a bounded extract or structural register exists.
  • Partial reviewed means only the displayed extract has been reviewed; it is not a completeness claim.
  • Source register records identity, authorship, structure or source status without presenting a verified complete text.
  • Navigation identifies a route or section guide rather than a reading edition.
  • This audit never manufactures missing verses or silently changes page text.
