Publication and recommendation boundary

  • Routes are shown only when they exist in the route directory and their effective canonical publication state is allowed.
  • Eight audited literature routes use the Release 231 boundary registry instead of a generic historical “published” label.
  • Draft, private, unpublished and archived states are excluded.
  • Counts are runtime calculations, not claims of complete Murugan coverage.
  • The route order uses local filters and deterministic sorting; it is not personalised AI.
  • No remote analytics, recommendation service or visitor profile is used.
