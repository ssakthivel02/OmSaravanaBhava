Print and PDF Support

Published pages now load a shared print layer. Use the page-level “Print / Save PDF” control, then choose your browser or operating system’s PDF destination when required.

Step 1 Open the required published page Use Search Facets, the reading list, Explore or the site directory to reach the exact route.
Step 2 Select “Print / Save PDF” The browser print dialog opens. Navigation, buttons, forms, audio controls and decorative shadows are removed from the printed copy.
Step 3 Choose paper or PDF Select a physical printer or a destination such as “Save as PDF” or “Microsoft Print to PDF”.
Step 4 Review before saving Confirm page range, paper size, orientation and Tamil text rendering before printing or creating the PDF.

Print output improvements

  • Print-only title and canonical-source header.
  • White background, black text and reduced visual decoration.
  • Navigation, forms, buttons, media players and update notices excluded.
  • Responsive images and printable tables.
  • Readable code wrapping and underlined links.
  • Existing Tamil and English content is not rewritten.

Important boundary

This release does not generate or host PDF files on a server. It provides clean print styling and invokes the browser print dialog. A browser-created PDF is a convenience copy of the displayed route; it does not change the route’s source status, review status, completeness or copyright position.

No claim is made that every browser, printer driver, font installation or assistive technology produces identical pagination.
